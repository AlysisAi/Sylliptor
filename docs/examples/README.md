# Examples

This directory contains small examples for documented extension points. They
are intentionally minimal and should be adapted before use in a real workspace.

## Contents

- `hooks/` contains command-hook examples and a sample hook configuration.
- `custom_tools/` contains a workspace custom-tool manifest example.

## Notes

Examples are not installed automatically. Review commands, environment
variables, file paths, and network behavior before trusting project-local hooks
or tools.
